# Copy Code Location

Copy code location (file:line) to clipboard.

## Usage

1. Select code in editor
2. Right-click → **Copy Code Location**
3. Paste the location (e.g., `@src/extension.ts#L10-17`)

## Development

```bash
npm install
npm run compile
# Press F5 to debug
```

## Installation

1. Package: `npx @vscode/vsce package` → install `.vsix`
2. Or copy to `~/.vscode/extensions/` and restart VSCode
